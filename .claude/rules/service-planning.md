# PeekTime 서비스 기획

## 온보딩 질문

### 1번 질문: 갑자기 쉬는 날이 생겼어요. 어떻게 보내고 싶으세요?
- 선택지: 밖에 나가서 뭔가 하고 싶어요 / 집이나 실내에서 편하게요

### 2번 질문: 계절 활동, 어떤 방식이 더 잘 맞아요?
- 선택지: 시간 내서 적극적으로 / 일상 안에서 부담 없이

### 3번 질문: 계절을 주로 어떻게 즐기는 편이에요?
- 선택지 (우선순위 결정): 자연이나 야외 활동 / 제철 음식이나 요리 / 감성 콘텐츠나 문화

---

## 사용자 타입 (1번 + 2번 질문 기반)

| 타입 | 공간 | 강도 |
|------|------|------|
| 자연 탐험가 | 밖 | 적극적 |
| 동네 산책러 | 밖 | 가볍게 |
| 제철 미식가 | 실내 | 적극적 |
| 일상 관찰자 | 실내 | 가볍게 |

---

## 미션 종류

### 1. 오늘의 미션
- 모든 사용자에게 동일하게 제공
- 미션풀 생성 → 배정대기 → 절기 내 모든 날에 미션 제공
- Admin에서 일자별 미션 드래그 앤 드롭으로 관리

### 2. 추천 미션
- 사용자 타입(4개)별로 LLM을 통해 생성
- 한 절기당 15개 (3번 질문 항목별 5개씩)
  - 자연이나 야외 활동: 5개
  - 제철 음식이나 요리: 5개
  - 감성 콘텐츠나 문화: 5개
- 사용자가 정한 우선순위에 맞게 위에서부터 순서대로 제공
- 사용자 타입 4개가 균등하게 생성되어야 함

### 3. 선택 미션
- 하루에 한 번만 생성 가능
- 필터링을 통해 선택
- **중요**: 오늘의 미션 15개 + 추천 미션 15개와 겹치면 안됨

---

## 태그 체계 (LLM 미션 생성 시 적용)

### 필터링용 태그 (선택미션 필터링)

| 구분 | 태그 |
|------|------|
| 공간 | 실내, 실외 |
| 강도 | 가벼운/이동없음/5분이내, 보통/동네범위/30분이내, 적극적/이동필요/1시간+ |
| 동반 | 혼자가능, 같이하면 더 좋음 |
| 카테고리 | 음식, 자연, 기록, 장소, 감각 |

### 추천미션 분류용 태그 (3번 질문 기반)
- 자연이나 야외 활동
- 제철 음식이나 요리
- 감성 콘텐츠나 문화

※ 이 태그는 선택미션 필터링이 아닌, 사용자 타입별 추천미션 5개씩 부여를 위한 용도

---

## 핵심 규칙 정리

1. **오늘의 미션**: 전체 사용자 동일, Admin에서 절기별 사전 설정
2. **추천 미션**: 사용자 타입별 15개, 우선순위순 정렬
3. **선택 미션**: 오늘의 미션 + 추천 미션과 중복 불가, 하루 1회 생성
