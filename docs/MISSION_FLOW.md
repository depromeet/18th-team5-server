## 개요

PeekTime은 24절기에 맞춰 사용자에게 미션을 제공하는 서비스입니다.

**핵심 아이디어**
- 사용자는 온보딩을 통해 자신의 타입이 결정됨
- LLM이 절기별 미션을 생성하고 태그를 붙임
- 관리자가 미션을 선별하여 사용자 타입별로 배정
- 절기 당일, 사용자는 자신의 타입에 맞는 미션을 받음

## 태그와 사용자 타입

### 태그 4개

| 태그 | 의미 | 예시 활동 |
|------|------|----------|
| #야외 | 밖에서 하는 활동 | 산책, 나들이, 사진 찍기 |
| #힐링 | 쉬면서 나를 돌보는 활동 | 명상, 차 마시기, 낮잠 |
| #창작 | 뭔가 만들고 표현하는 활동 | 그림, 글쓰기, 요리 |
| #소통 | 사람들과 함께하는 활동 | 연락하기, 대화, 모임 |

### 사용자 타입 6개

사용자는 온보딩을 통해 **태그 2개 조합**으로 타입이 결정됩니다.

| 타입 번호 | 태그 조합 | 타입 이름 (예시) |
|----------|----------|-----------------|
| 타입 1 | 야외 + 힐링 | 자연 힐러 |
| 타입 2 | 야외 + 창작 | 감성 탐험가 |
| 타입 3 | 야외 + 소통 | 액티브 커넥터 |
| 타입 4 | 힐링 + 창작 | 집순이 아티스트 |
| 타입 5 | 힐링 + 소통 | 따뜻한 상담사 |
| 타입 6 | 창작 + 소통 | 표현하는 사람 |

## STEP 1: 온보딩 - 사용자 타입 분류

### 목적
질문을 통해 사용자의 성향을 파악하고, 6개 타입 중 하나로 분류합니다.

### 온보딩 질문 (5개)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│                        나에게 맞는 미션을 찾아볼게요!                      │
│                            (약 1분 소요)                                 │
│                                                                         │
│                           [시작하기]                                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Q1. 갑자기 반차가 생겼어요. 뭐 할래요?**

| 선택지 | 점수 |
|--------|------|
| A) 날씨 좋으면 밖에 나가야지 | 야외 +1 |
| B) 집에서 낮잠이나 자야지 | 힐링 +1 |
| C) 밀린 취미 생활 해야지 | 창작 +1 |
| D) 친구한테 연락해봐야지 | 소통 +1 |

**Q2. 카페 가면 주로 뭐 해요?**

| 선택지 | 점수 |
|--------|------|
| A) 창가 자리에서 바깥 구경 | 야외 +1 |
| B) 조용히 멍 때리거나 책 읽기 | 힐링 +1 |
| C) 노트북이나 다이어리 꾸미기 | 창작 +1 |
| D) 친구랑 수다 떨기 | 소통 +1 |

**Q3. 기분 전환이 필요할 때?**

| 선택지 | 점수 |
|--------|------|
| A) 동네 산책이나 드라이브 | 야외 +1 |
| B) 따뜻한 물로 샤워하거나 목욕 | 힐링 +1 |
| C) 음악 듣거나 뭔가에 몰두 | 창작 +1 |
| D) 누군가한테 전화하기 | 소통 +1 |

**Q4. 여행지에서의 나는?**

| 선택지 | 점수 |
|--------|------|
| A) 아침부터 돌아다니는 편 | 야외 +1 |
| B) 숙소에서 여유 즐기는 편 | 힐링 +1 |
| C) 사진 예쁘게 찍는 게 중요 | 창작 +1 |
| D) 현지인이나 동행과 대화 | 소통 +1 |

**Q5. 나랑 더 가까운 쪽은?**

| 선택지 | 점수 |
|--------|------|
| A) 집보다 밖이 좋아 | 야외 +1 |
| B) 혼자만의 시간이 필요해 | 힐링 +1 |
| C) 뭔가 만드는 게 좋아 | 창작 +1 |
| D) 사람들이랑 있는 게 좋아 | 소통 +1 |

### 점수 계산

```
[예시 답변]
Q1: A (야외 +1)
Q2: C (창작 +1)
Q3: A (야외 +1)
Q4: C (창작 +1)
Q5: B (힐링 +1)

[최종 점수]
야외: 2점 ⭐
힐링: 1점
창작: 2점 ⭐
소통: 0점

→ 상위 2개 = 야외, 창작
→ 내 타입: "야외+창작" (감성 탐험가)
```

### 동점 처리

만약 상위 점수가 동점이면 추가 질문:


## STEP 2: LLM 미션 생성

### 목적
24절기에 맞는 미션을 LLM이 생성합니다. 각 미션에는 태그 2개가 붙습니다.

### 생성 시점
절기 7일 전에 배치(자동 스케줄러)로 실행

### LLM 프롬프트 예시

```
당신은 절기 기반 미션을 생성하는 전문가입니다.

[절기 정보]
- 절기: 입춘
- 시기: 2월 4일경
- 의미: 봄의 시작, 새로운 시작의 기운

[태그 설명]
- #야외: 밖에서 하는 활동 (산책, 나들이, 사진)
- #힐링: 쉬면서 나를 돌보는 활동 (명상, 차, 휴식)
- #창작: 만들고 표현하는 활동 (그림, 글, 요리)
- #소통: 사람들과 함께하는 활동 (연락, 대화, 모임)

[요청]
입춘에 어울리는 일상 미션 60개를 만들어주세요.
각 미션에 태그 2개를 붙여주세요.
6가지 태그 조합이 골고루 포함되도록 해주세요.

[태그 조합 6가지]
1. 야외+힐링
2. 야외+창작
3. 야외+소통
4. 힐링+창작
5. 힐링+소통
6. 창작+소통

[출력 형식]
1. 미션 내용 [#태그1 #태그2]
2. 미션 내용 [#태그1 #태그2]
...
```

### LLM 결과 예시

```
[입춘 미션 60개]

1. 동네 공원에서 봄 기운 느끼며 산책하기 [#야외 #힐링]
2. 봄꽃 찾아서 사진 찍기 [#야외 #창작]
3. 친구와 함께 봄나들이 가기 [#야외 #소통]
4. 창문 열고 봄바람 맞으며 차 마시기 [#힐링 #창작]
5. 가족에게 봄 안부 전화하기 [#힐링 #소통]
6. 봄을 주제로 짧은 글 쓰고 공유하기 [#창작 #소통]
7. 해 뜨는 시간에 일어나 아침 산책하기 [#야외 #힐링]
8. 봄 풍경 스케치하기 [#야외 #창작]
...
```

## STEP 3: 어드민에서 미션 선별 및 배정

### 목적
LLM이 생성한 미션을 검토하고, 6개 사용자 타입에 맞게 배정합니다.

### 플로우

```
[1] 태그로 미션 필터링해서 보기
              │
              ▼
[2] 원하는 미션 선택
              │
              ▼
[3] 선택한 미션을 사용자 타입에 배정
              │
              ▼
[4] 타입별 현황 확인 & 저장
```
