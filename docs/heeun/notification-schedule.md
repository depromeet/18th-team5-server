# 푸시 알림 스케줄 정책

## 알림 종류

| 알림 | 발송 시간 | 이동 위치 |
|------|----------|----------|
| 오늘의 미션 알림 | 매일 오전 10시 | 홈 탭 |
| 절기 시작 알림 | 절기 시작일 오전 8시 | 절기소개 탭 |
| 절기 종료 알림 | 절기 마지막 날 오후 8시 | - |

---

## 1. 오늘의 미션 알림

**발송 조건**: 사용자가 오늘의 미션 알림을 ON으로 설정한 경우

| 항목 | 내용 |
|------|------|
| **title** | {절기명}의 오늘의 미션이 도착했어요! |
| **body** | 지금 바로 확인하고 제철을 즐겨보세요 🌿 |

**예시**
```
망종의 오늘의 미션이 도착했어요!
지금 바로 확인하고 제철을 즐겨보세요 🌿
```

---

## 2. 절기 시작 알림

**발송 조건**: 사용자가 절기 시작 알림을 ON으로 설정한 경우

| 항목 | 내용 |
|------|------|
| **title** | 새로운 절기, {절기명}이 시작되었어요! |
| **body** | 이번 절기에는 어떤 이야기가 담겨있을까요? 🌿 |

**예시**
```
새로운 절기, 망종이 시작되었어요!
이번 절기에는 어떤 이야기가 담겨있을까요? 🌿
```

---

## 3. 절기 종료 알림

**발송 조건**: 사용자가 절기 종료 알림을 ON으로 설정한 경우

| 항목 | 내용 |
|------|------|
| **title** | {절기명}의 마지막 날이에요 |
| **body** | 이번 절기에 완료하지 못한 미션이 있다면 지금 확인해보세요! 🍂 |

**예시**
```
망종의 마지막 날이에요
이번 절기에 완료하지 못한 미션이 있다면 지금 확인해보세요! 🍂
```

---

## 스케줄러 Cron 표현식

| 알림 | Cron | 설명 |
|------|------|------|
| 오늘의 미션 | `0 0 10 * * *` | 매일 10:00 |
| 절기 시작 | `0 0 8 * * *` | 매일 08:00 (시작일만 발송) |
| 절기 종료 | `0 0 20 * * *` | 매일 20:00 (종료일만 발송) |

> 타임존: `Asia/Seoul`
